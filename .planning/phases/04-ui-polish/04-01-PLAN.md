<task type="auto">
  <name>Task 1: Create MainViewModel with @Observable</name>
  <files>Prunr/ViewModels/MainViewModel.swift</files>
  <action>
Create @Observable class MainViewModel with:
- `snapshots: [Snapshot]` - all available snapshots from DB
- `selectedBeforeSnapshot: Snapshot?` - older snapshot for comparison
- `selectedAfterSnapshot: Snapshot?` - newer snapshot for comparison
- `deltas: [Delta]` - current comparison results
- `isScanning: Bool` - scan in progress flag
- `scanProgress: String` - current path being scanned
- `errorMessage: String?` - user-visible error

Methods:
- `loadSnapshots()` async - fetch from DatabaseManager.fetchAllSnapshots()
- `scan(path: String)` async - call ScanService.scan, update progress, reload snapshots
- `compareSnapshots()` async - call DeltaService.compare with selected IDs
- `autoSelectSnapshots()` - select two most recent snapshots if available

Use @MainActor for UI state. Handle errors gracefully with errorMessage. Avoid force unwrapping - use guard/if-let.
  </action>
