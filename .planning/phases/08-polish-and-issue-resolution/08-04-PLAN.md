<task type="auto">
  <name>Task 4: Fix multi-monitor popup positioning (ISS-025)</name>
  <files>Prunr/Services/MenuBarManager.swift</files>
  <action>
    Ensure popup stays near menubar icon, not jumping to second monitor:

    **Current implementation:**
    ```swift
    popover?.show(relativeTo: button.bounds, of: button, preferredEdge: .minY)
    ```

    **Issue:**
    When focus changes, popup might reposition to another screen.

    **Root cause:**
    - NSPopover.show() positioning might not anchor properly
    - Window focus events trigger repositioning
    - Multiple monitors cause coordinate confusion

    **Fix approach:**

    **1. Lock popup to menubar screen:**
    ```swift
    func togglePopover() {
        guard let button = statusItem?.button,
              let buttonWindow = button.window else { return }

        if let popover = popover, popover.isShown {
            popover.performClose(nil)
            isPopoverShown = false
        } else {
            // Get menubar screen
            if let screen = buttonWindow.screen {
                // Calculate position relative to menubar icon
                let buttonFrame = button.convert(button.bounds, to: nil)
                let screenButtonFrame = buttonWindow.convertToScreen(buttonFrame)

                // Show popover
                popover?.show(relativeTo: button.bounds, of: button, preferredEdge: .minY)

                // Ensure popup window stays on correct screen
                if let popoverWindow = popover?.contentViewController?.view.window {
                    popoverWindow.setFrameTopLeftPoint(NSPoint(
                        x: screenButtonFrame.midX - popoverWindow.frame.width / 2,
                        y: screenButtonFrame.minY - 8
                    ))
                }
            }
        }
    }
    ```

    **2. Prevent popup from moving:**
    - Set popover.behavior = .transient (already done)
    - Monitor window move notifications
    - Reset position if popup moves to different screen

    **3. Handle focus changes:**
    - Popup should close on focus loss (.transient behavior handles this)
    - If popup stays open, anchor position should not change

    **Testing:**
    Requires multi-monitor setup:
    - Click menubar icon on primary monitor
    - Verify popup appears on primary monitor
    - Switch focus to window on secondary monitor
    - Verify popup doesn't jump to secondary monitor
    - Test with different monitor arrangements (side-by-side, above/below)
  </action>
