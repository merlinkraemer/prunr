  <name>Task 2: Create DeltaService with comparison logic</name>
  <files>Prunr/Services/DeltaService.swift</files>
  <action>Create DeltaService actor with:
  - static let shared = DeltaService()
  - func compare(beforeId: Int64, afterId: Int64) async throws -> [Delta]

  Algorithm:
  1. Fetch entries for both snapshots via DatabaseManager.shared.fetchEntries()
  2. Build Dictionary mapping path -> size for each snapshot
  3. Build union of all paths (both snapshots)
  4. For each path, create Delta with:
     - oldSize = beforeDict[path] (nil if not present)
     - newSize = afterDict[path] (nil if not present)
     - change = (newSize ?? 0) - (oldSize ?? 0)
  5. Filter: only include items where change != 0
  6. Sort: descending by changeBytes (largest growth first, shrinkage at bottom)

  Use actor isolation (not class) for thread safety. DO NOT access database directly - use DatabaseManager methods.

  Optimization hints:
  - Dictionary lookup is O(1) vs array search O(n)
  - Pre-allocate array capacity when possible
  - No Task.yield() needed - comparison is CPU-bound and fast</action>
  <verify>swiftc -parse succeeds on DeltaService.swift, no direct DatabasePool access (only via DatabaseManager)</verify>
  <done>DeltaService actor with compare() method that returns sorted [Delta] with changes calculated correctly</done>
