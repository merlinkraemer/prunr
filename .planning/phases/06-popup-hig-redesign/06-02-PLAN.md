**HIG Standards to Apply:**
- **Preferences windows:** 20pt margin on left, right, bottom; 12-24pt spacing between controls
- **Group boxes:** 12pt above and below separators for Regular controls
- **About windows:** 400×260pt minimum window size, 80pt icon, prominent app name, version info, copyright
- **List/table views:** 24pt (small) or 32pt (large) row heights, 10pt vertical inset for inset-style tables
- **Control sizing:** Regular size default, consistent within same container

**Issues being addressed:**
- Current SettingsView frame (450×400) is larger than needed but acceptable
- About tab should follow HIG about window pattern more closely
- All tabs should use consistent HIG spacing
</context>
