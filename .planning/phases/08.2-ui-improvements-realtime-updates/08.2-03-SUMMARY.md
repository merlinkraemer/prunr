## Technical Details

### Time-Based Progress Estimation (ISS-033)

Since FileScanner streams results without knowing the total upfront, a time-based estimation approach was implemented:

- 0-2 seconds: 0% (no progress bar shown)
- 2-5 seconds: Ramp to 25%
- 5-10 seconds: Ramp to 50%
- 10-20 seconds: Ramp to 75%
- 20+ seconds: Gradually approach 95%

This provides visual feedback without the performance cost of pre-enumerating files.

### Settings Tab Navigation (ISS-034)

Modified SettingsView to read initial tab from UserDefaults:
```swift
@State private var selectedTab = UserDefaults.standard.integer(forKey: "settingsSelectedTab")
```

MenuBarView sets the tab before opening settings:
```swift
UserDefaults.standard.set(1, forKey: "settingsSelectedTab") // 1 = Paths tab
```
