### Task 1: Parallel Path Scanning

In `createBaselines()` (MenuBarManager ~line 617):

- Currently scans each tracked path sequentially in a for loop
- Change to use `TaskGroup` for independent paths (paths that don't overlap)
- `effectiveTrackedPaths` already deduplicates nested paths, so remaining paths are independent

### Task 2: Live Category Fill-In During Scan

In `ScanService.scan()`:

- The `categoryTotals` dict is already accumulated in-memory during scan

- Categories appear and grow in real-time as the scan progresses
