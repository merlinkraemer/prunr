# Plan: UI & UX Quick Fixes

## Objective

Fix three UI/UX issues identified during testing: reset baseline placement, tree view visuals, and app icon configuration.

## Execution Context

**Critical files:**
- `Prunr/Views/SettingsView.swift` - Reset baseline button placement
- `Prunr/Views/CategoryGrowthListView.swift` - NestedBigItemRow visual design
- `Prunr/PrunrMenuBar.swift` - Dock icon policy
- `Prunr/Assets.xcassets/AppIcon.appiconset/` - App icon assets

## Context

### ISS-061: Reset Baseline Placement
The Reset Baseline button is currently in the About tab, but it logically belongs in the Paths tab since it's related to path/baseline management.

### ISS-062: Tree View Visuals
Nested big file items currently have the same visual treatment as parent items (background, border, hover state). They should look like nested children, not separate items.

### ISS-063: App Icon
App icon is not configured - needs to be added from the CI directory.

## Tasks

### Task 1: Move Reset Baseline to Paths Settings Tab (ISS-061)
**File:** `SettingsView.swift`

1. Remove Reset Baseline button from AboutSettingsTab
2. Add Reset Baseline button to PathsSettingsTab footer
3. Keep the same functionality (resetBaseline() call)

### Task 2: Redesign NestedBigItemRow Visuals (ISS-062)
**File:** `CategoryGrowthListView.swift`

1. Remove background color from nested items
2. Remove border/stroke from nested items
3. Change tree connector color from orange to secondary gray
4. Change file icon from orange to secondary color
5. Keep simple indentation and connector lines for hierarchy

### Task 3: Add App Icon (ISS-063)
**Files:** `Prunr/Assets.xcassets/AppIcon.appiconset/`

1. Generate icon sizes from source: `ci/logo/prunr-iOS-Default-1024x1024@1x.png`
2. Create all required sizes: 16x16, 32x32, 128x128, 256x256, 512x512 (1x and 2x)
3. Update Contents.json with filename references

### Task 4: Ensure No Dock Icon (UX Enhancement)
**File:** `PrunrMenuBar.swift`

Add `NSApp.setActivationPolicy(.accessory)` in init() to programmatically ensure dockless operation (in addition to LSUIElement).
