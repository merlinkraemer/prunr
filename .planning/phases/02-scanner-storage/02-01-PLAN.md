  <action>Create FileScanner as an actor with:

**Properties:**
- `private let fileManager = FileManager.default`
- `private var visitedInodes = Set<NSNumber>()` for hard link deduplication

**Main method:**
- `func scan(_ rootURL: URL) -> AsyncStream<ScanResult>` returns streaming results

**Implementation MUST follow research patterns:**
- Use `FileManager.default.enumerator(at:includingPropertiesForKeys:options:errorHandler:)`
- Resource keys: `.isRegularFileKey`, `.totalFileAllocatedSizeKey`, `.isSymbolicLinkKey`, `.isAliasFileKey`, `.fileResourceIdentifierKey`
- Options: `[.skipsHiddenFiles, .skipsPackageDescendants]`
- Error handler: captures permission-denied paths, returns true to continue

**Safety checks in shouldSkip(_ url:) helper:**
- Skip if `isSymbolicLink == true` (prevents infinite loops)
- Skip if `isAliasFile == true`
- Track inodes via `fileResourceIdentifierKey` - skip if already visited

**Size calculation:**
- Use `totalFileAllocatedSizeKey` for actual APFS disk usage
- Fallback to `fileAllocatedSizeKey`, then `fileSizeKey`
- Cast result to Int64 for storage

**Streaming pattern:**
- Yield each result via `continuation.yield()`
- Call `await Task.yield()` every 1000 items to prevent blocking
- Call `continuation.finish()` when enumeration completes

**CRITICAL:** Do NOT use string-based enumeration or fileSizeKey - these are deprecated/incorrect for APFS.</action>
